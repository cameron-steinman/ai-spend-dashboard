Data Sources & Pricing

How Token Data Is Collected

Every API call from Anthropic and Google logs a usage object in session JSONL files at ~/.openclaw/agents/main/sessions/*.jsonl. Each assistant message includes input, output, cacheRead, and cacheWrite token counts, plus a pre-calculated cost object from OpenClaw's billing layer. Ingested via ingest-usage.py, exported via generate-report.py.

Perplexity (web_search): Tool calls don't emit token data. Cost estimated at $0.005/call. ESTIMATED

Groq, Ollama, OpenClaw internal, OpenRouter free: Logged but cost = $0.

Cost Calculation

Costs for Anthropic and Google are calculated by OpenClaw's billing layer at response time and stored in session JSONL. This dashboard reads those pre-calculated values. Cache read tokens billed at ~10% of input rate. Google implicit caching (auto since May 2025) already reflected in amounts.

Audit Column

The Audit column in the daily table cross-references JSONL-derived costs against provider billing APIs:

✅ = Anthropic Admin API confirmed (delta <5%)

⚠️ = Mismatch detected between JSONL and billing API

⚡ = JSONL-only (no billing API available — Google, OpenRouter, etc.)

Timezone

All dates and hours are displayed in Eastern Time (EST/EDT). The system auto-adjusts for daylight saving time using America/Toronto.
